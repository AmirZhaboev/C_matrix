# s21_matrix

Библиотека для работы с матрицами на языке **C**.

Проект включает реализацию базовых операций линейной алгебры: создание матриц, арифметические операции, транспонирование, вычисление определителя, матрицы алгебраических дополнений и обратной матрицы.

---

## Реализованный функционал

### Создание и удаление матриц

- `s21_create_matrix`
- `s21_remove_matrix`

---

## Сравнение матриц

- `s21_eq_matrix`

Проверка матриц на равенство.

---

## Арифметические операции

- `s21_sum_matrix`
- `s21_sub_matrix`
- `s21_mult_number`
- `s21_mult_matrix`

---

## Преобразования матриц

- `s21_transpose`

---

## Продвинутые операции

- `s21_determinant`
- `s21_calc_complements`
- `s21_inverse_matrix`

## Пример использования

```c
matrix_t A;
matrix_t B;
matrix_t result;

s21_create_matrix(3, 3, &A);
s21_create_matrix(3, 3, &B);

s21_sum_matrix(&A, &B, &result);
```
